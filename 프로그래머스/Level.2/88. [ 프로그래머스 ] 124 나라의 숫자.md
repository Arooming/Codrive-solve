> 
> **난이도**: 2
>
> **유형**: #수학


### [ 문제 풀이 ]
```JavaScript
function solution(n) {
  let answer = "";
  const arr = [4, 1, 2];

  while (n > 0) {
    answer = arr[n % 3] + answer;
    n = Math.floor((n - 1) / 3);
  }

  return answer;
}
```
