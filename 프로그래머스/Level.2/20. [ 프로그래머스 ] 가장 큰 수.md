> **난이도**: 2
>
> **유형**: #정렬


### [ 문제 풀이 ]
```JavaScript
function solution(numbers) {
    const strArr = numbers.map((num) => num.toString());
    const answer = strArr.sort((a, b) => b + a - (a + b)).join("");
    
    return answer[0] === "0" ? "0" : answer;
}
```
